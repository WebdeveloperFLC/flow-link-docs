# FAQ seed — 30 Q&A per service

Generated by:

```bash
node scripts/expand-service-faqs.mjs
node scripts/generate-faq-sql.mjs
```

## Apply in Supabase SQL Editor

**Easiest:** paste and run **`01-apply-all-faqs.sql`** once (all 36 services, ~150 KB).

Or run individual `*.sql` files (~4 KB each).

1. Run `01-apply-all-faqs.sql` (or each service file)
2. Run `00-diagnose-faq-counts.sql` — target: **done_30 = 36, need_more = 0**

## Or upload from terminal

```bash
export VITE_SUPABASE_URL=...
export SUPABASE_SERVICE_ROLE_KEY=...
node scripts/upload-all-faqs.mjs
```

## Verify one service

```sql
SELECT jsonb_array_length(academy_metadata->'faqs') AS faq_count
FROM service_library
WHERE id = 'b2000001-0001-4000-8000-000000000054';  -- Germany Opportunity Card
```

Expect: **30**

Does **not** overwrite quiz questions (uses `jsonb_set` on `faqs` only).
