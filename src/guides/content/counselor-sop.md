# Counselor SOP

> **Status:** Draft — expand with Future Link counselor procedures.

Standard operating procedures for **counselors** using the CRM day to day.

---

## 1. Daily workflow

- [ ] Review assigned clients and tasks
- [ ] Check notification bell and handoffs
- [ ] Respond to portal chat and WhatsApp (assigned threads only)
- [ ] Update client status and timeline notes

---

## 2. Client access

- Counselors see clients via **assignment**, **client_access**, or team defaults
- Permissions: view/edit/upload per `can_edit_client` — see Settings → Users if blocked

---

## 3. Key modules

| Task | Where in CRM |
|---|---|
| Client record | Clients → open client |
| Documents | Client → Documents |
| Invoices / payments | Client → Invoices (draft create where permitted) |
| Portal invite | Client → Access / Invite |
| WhatsApp helpline | WhatsApp (assigned threads) — [WhatsApp Usage Guide](./whatsapp-helpline.md) |
| Leads | Leads (if role includes lead access) |

---

## 4. Escalation

| Situation | Action |
|---|---|
| Cannot edit client | Ask admin to check role + client access |
| Payment / receipt issue | Follow invoice SOP; escalate to accounting |
| Telephony failure | [TeleCMI Usage Guide](./telecmi-usage-guide.md) |
