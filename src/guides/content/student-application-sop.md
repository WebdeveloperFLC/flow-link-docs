# Student Application SOP

> **Status:** Draft — expand with Future Link application workflow by country/program.

End-to-end process for managing a **student application** from intake through submission readiness.

---

## 1. Scope

Covers CRM workflow for counseling staff — not embassy or institution portals unless linked below.

---

## 2. Typical stages

```flow
Lead or walk-in
  → Convert to client
  → Collect documents
  → Complete questionnaires / forms
  → Internal review
  → Application submitted (status + timeline)
  → Visa filing (see Visa Filing SOP)
```

---

## 3. CRM steps

| Step | Action |
|---|---|
| Create / convert | Leads → Convert, or Clients → New |
| Profile & education | Client profile tabs |
| Documents | Upload to `client-documents`; classify if using AI pipeline |
| Questionnaires | Client → Forms — send link; track `submitted` status |
| Status updates | Set client status; timeline logs `status_change` automatically |
| Team | Assign counselor; add client_access for documentation staff |

---

## 4. Checklists (customize per program)

- [ ] Identity documents collected
- [ ] Academic records collected
- [ ] Questionnaire submitted by client
- [ ] Internal QA complete
- [ ] Application marked submitted in CRM
