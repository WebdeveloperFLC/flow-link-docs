# Data Uploading

- **Import cold leads** (CSV): `/leads/cold` → Import. Required: name, phone, email; optional: country code, source, interested country/service. Phones normalized; duplicates flagged.
- **Client documents**: client → Documents → drag-and-drop. Supports PDF, images, Word, Excel. Large PDFs split by section.
- **Digital Success Hub**: `/digital-success` → Upload — accepts links (Drive/OneDrive) or direct files.
- **Accounting documents**: `/accounting/documents/upload` — bills/invoices; OCR at `/accounting/documents/ocr`.
- **Course data**: admins import course catalogues via Institutions → Course Review.

Tip: when import fails, check the error report — it lists row-by-row reasons.
