# Clients & Client Entry

**Create a client**: sidebar → **+ New Client** (`/clients/new`). Enter personal details, contact, interested country/service, branch, assigned counselor.

**Client detail page** (`/clients/:id`) sections:
- **Overview** — basic info, status, assigned team.
- **Timeline** — chronological activity (calls, emails, notes, file uploads).
- **Documents / Binder** — uploaded files grouped by section (passport, education, finance).
- **Forms / Questionnaire** — send a questionnaire link; responses appear here.
- **Chat** — internal staff chat and staff↔client chat.
- **Assessment** — Settle Abroad eligibility results.
- **Letters** — generate letters from templates.
- **Offers** — apply promotional offers/discounts.

**Uploading documents**: client → Documents tab → drag-and-drop or upload zone. Files are auto-classified. Large PDFs can be split by section.

**Sharing a client**: Team Access grants another staff member access to a specific client.

**Portal invitation**: from client detail, invite the client to the client portal (`/portal`) to upload documents, chat, view offers, track applications.
