# Icons

Performance Hub uses **lucide-react** icons declared in `ROUTES/performanceWorkspaceNav.ts` and page imports.

There is no local SVG icon set for hub modules. Module accents use CSS tokens (`--cash`, `--wallet`, `--offer`) in `STYLES/src__styles__performance-hub-theme.css`.

Brand logo colors: `--flc-blue`, `--flc-red` in `STYLES/src__index.css`.
